# PasarIkan App 🐟

## 📖 About

PasarIkan App is a mobile application concept designed to connect local fish sellers with customers through a simple digital platform. The application aims to make buying and selling fresh fish easier, faster, and more accessible for local communities.

This project was created as a learning project while exploring Android application development and user interface design.

---

## 🎯 Objectives

- Help local fish sellers reach more customers.
- Simplify the fish ordering process.
- Promote digital transformation for small fisheries.
- Provide a user-friendly mobile experience.

---

## ✨ Features

- User Registration & Login
- Browse Fish Products
- Product Details
- Search Products
- Shopping Cart
- Order Management
- User Profile

> Features may vary depending on the current development stage.
